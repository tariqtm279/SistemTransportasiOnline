## 📖 Overview

**SistemTransportasiOnline** is a Java application that simulates an online transportation service. Developed as part of an **Object-Oriented Programming** course, the project demonstrates the implementation of core OOP concepts while managing transportation-related entities such as drivers, passengers, vehicles, trips, payments, and vouchers.

The application provides two execution modes:

* **CLI (Command-Line Interface)** for console-based interaction.
* **GUI (Graphical User Interface)** built with Java Swing for a more interactive user experience.

## 🚀 Getting Started

Clone the repository.

```bash
git clone https://github.com/<username>/<repository-name>.git
```

Navigate to the project directory.

```bash
cd <repository-name>
```

Compile the project.

```bash
javac src/*.java
```

### Run the CLI Version

```bash
java src.Main
```

### Run the GUI Version

```bash
java src.TransportasiGUI
```
